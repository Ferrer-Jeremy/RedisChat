# RedisChat

ChatRedis enable you to chat on a redis server

![Screenshot](RedisChat.png)

### Get It Working
1. Go in src
2. Launch a terminal and type 

```bash
python3 mainWindow.py
```
3. Connect to a server
4. Add a channel
5. Start chating
